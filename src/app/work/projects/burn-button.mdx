## Overview

An AI-powered automation agent that streamlines email management and tasks via messaging apps.

## Key Features

- **AI-Powered Automation**: Automates email replies and drafting using AI agents.
- **Telegram Integration**: Integrates directly with Telegram for on-the-go command execution.

## Technologies Used

- **n8n**: For workflow automation.
- **OpenAI API**: For intelligent text generation and processing.
- **Telegram API**: For the messaging interface.
